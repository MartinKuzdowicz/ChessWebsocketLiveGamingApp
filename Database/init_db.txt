with mongolab mongodb

You can configure a local db conncetion in this steps:
1. go to MongoDBConnectionConfig.java
2. change the linie:

private String mongoLabConnectionString = ChessAppProperties
			.getProperty("db.conn.production");
			
into:

private String mongoLabConnectionString = ChessAppProperties
			.getProperty("db.conn.localhost");

3. Create local database on local mongodb server with below commands in mongo shell


use chessapp_db
